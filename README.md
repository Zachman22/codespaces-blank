# Codespaces Beautiful Starter

This repository contains a polished starter site with full developer tooling configured:

- Vite + React
- Tailwind CSS
- ESLint + Prettier
- Vitest unit testing
- Playwright setup for E2E tests
- GitHub Actions CI workflow
- Devcontainer for Codespaces / VS Code Remote - Containers
- VS Code recommended extensions

Quick start (locally)
1. npm install
2. npm run dev
3. Open http://localhost:3000

Devcontainer / Codespaces
- Open in Codespaces or open this repo in VS Code and "Reopen in Container". The devcontainer will set up Node and recommended tools.

CI
- The GitHub Actions workflow runs lint, tests, and build on pushes and PRs.
